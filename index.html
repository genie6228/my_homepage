AI 상세페이지

고객의 제품과 브랜드 특징을 고려하여, 주목도 높은 쇼츠, 릴스를 제작합니다.

AI 추천 알고리즘을 최적화하여, 주목도, 전환율을 높일수 있는 쇼츠,릴스입니다.

기간: 난이도에 따라, 1주일~2주일

가격:

  • 단순 편집 : 30~50만
  • 기획 + 디자인 + 편집 : 50~100만
  • 기획 + 디자인 + 영상 효과 : 100~200만

촬영이 필요할 경우, 추가 기간, 비용 발생

웹페이지, 랜딩페이지

AI, 인공지능 최적화 웹페이지 기획, 디자인, 배포

AI 기반 전략으로 웹페이지 성과를 극대화하세요.

기획부터 디자인, 배포까지 전 과정 원스톱 지원!

방문자 행동을 분석해 전환율을 높입니다. 당신의 웹페이지,랜딩페이지를 이제 인공지능에 최적화 하십시오.

기간: 난이도에 따라, 1개월~3개월

가격:

  • 브랜드, 제품, 예약, 구매, 챗봇 : 500만
  • 브랜드, 제품, 예약, 구매, 챗봇 + 로그인인증 : 700만
  • 브랜드, 제품, 예약, 구매, 챗봇 + 로그인인증 + DB + 고객관리 : 1000만

브랜드 Naming Package

현존 최고수준의 브랜드 Name Package

창업자의 철학을 담아 직관적이고 감각적인 네이밍을 만들어드립니다.

스시도쿠 겐, 견체공학, 헤겔의 빛 등 잊을수 없는 브랜드 이름을 지어드립니다.

브랜드 이름, 슬로건, 약속, 페르소나등 브랜드전략의 기초를 만들어드립니다.

포함 서비스:

  • 브랜드 이름 대안 3가지 제시
  • 고객 페르소나 정의
  • 슬로건, 핵심 메시지, 브랜드 스토리 개발

기간: 약 1개월 소요

가격: 500만원 (중간계 기존 수강생은 30% 할인)
